
A precompiled version of this program has been placed in the binn folder in this project.
You can open a command prompt and navigate to that directory and run the program.  Note
that the executable takes three paramaters.

FIRST - URL endpoint of the Event Grid Topic such as 
    https://eventgridtoplab3.westus2-1.eventgrid.azure.net/api/events
SECOND - sasKey for the Event Grid Topic such as 
    robDsntahoeBRntahoehaeoP/1zeVxvMYwQqoA=
THIRD - the number of times to repeat sending the event
